# "一生一芯"工程项目

这是"一生一芯"的工程项目。
其中`npc`文件是处理器Verilog源码及Verilator仿真环境，均为本人编写。
`ysyx-exam`是流片考核环境设置脚本。
`doc`是仙剑运行截图。
其余文件及本人编写的部分均与PA一致（NEMU为RV64版本，AM中额外适配了NPC相关的源代码及Makefile脚本支持）

具体请参考[一生一芯实验讲义][lecture note]，和[PA讲义][pa]
